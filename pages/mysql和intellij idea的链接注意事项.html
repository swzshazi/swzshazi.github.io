mysql和intellij idea的链接注意事项

  1. 首先需要配置maven中的依赖项，即在pom.xls中配置添加mysql_connector_java.jar依赖库，具体添加方式略， 添加时库的详细信息可百度搜索“mvn仓库（或者Maven Central Repository Search）”，然后进入仓库搜索“mysql”即可，如下图所示。

  2. 当使用localhost登陆时，当前用户名可以用"select host, user from mysql.user;"命令查询，连接mysql时即采用查询到的用户名。值得注意的是输入的命令后面有一个“;”，不能省略。

    1. 首先，键盘输入"win+R"打开运行，输入"cmd"打开命令提示符。
    2. 连接mysql

    3. 输入命令"select host, user from mysql.user;"如下图所示，此时红框内就是对应的用户名

  3. mysql操作命令中当涉及到表单名称时，需要加反引号将表单名称框出，以插入数据为例，如下所示，1中的写法就会报错，而2就会正确执行。

    1. insert into table1 values(a, b, c, d);
    2. insert into `table1` values(a, b, c, d);
